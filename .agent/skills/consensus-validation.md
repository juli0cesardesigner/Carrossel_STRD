# Skill: Consensus Validation

Esta habilidade define o protocolo de validação dentro da Sala de Guerra.

## Protocolo de Decisão
1. **AVALIAÇÃO**: O agente deve ler a proposta do Narrador.
2. **FILTRO**: O agente aplica os filtros de seu arquivo `knowledge/*.md`.
3. **DECISÃO**: O agente deve responder EXCLUSIVAMENTE com uma destas opções:
   - **[APROVADO]**: A ideia passa pelos filtros. O agente pode adicionar um pequeno elogio ou sugestão de melhoria (opcional).
   - **[VETO]**: A ideia falhou em um critério crítico. O agente DEVE explicar o motivo e sugerir uma direção para correção.

## Critério de Unanimidade
Uma ideia só é considerada "Final" quando houver 5 marcações de **[APROVADO]**.
